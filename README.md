# Gujikmon 
진행 기간 : 2021.3.20 ~ 2021.4.26

주제 : 구직자들을 위한 위치 기반 강소기업 검색 사이트

팀명 : DevOps (4조)

팀원 : 정산하, 김성수, 고재경, 이반석


### 주요 스킬 셋

언어 : Python3, javaScript(ES6)
프레임워크 : Django, React
DB : MongoDB
서버 : AWS EC2
